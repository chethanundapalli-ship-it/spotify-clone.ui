# spotify-clone.ui
A simple Spotify Clone built using HTML, CSS, and JavaScript. It features a responsive music player interface with play, pause, and basic UI functionality inspired by Spotify.
Spotify Clone

A responsive Spotify Clone built using HTML5 and CSS3. This project recreates the look and feel of the Spotify web player with a clean, modern interface. It demonstrates responsive web design, structured layouts, and reusable UI components.
